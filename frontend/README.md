# CampusConnect Frontend

React + Vite frontend for CampusConnect.

## Tech

- React + Vite
- React Router
- Tailwind CSS
- Axios (JWT interceptor)
- Framer Motion
- React Hook Form

## Run

```bash
npm install
npm run dev
```

The dev server proxies backend API calls (no CORS needed):
- `/api` -> `http://localhost:8080`
